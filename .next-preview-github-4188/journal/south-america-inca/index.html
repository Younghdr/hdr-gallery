南美：印加傳奇（上）

秘魯、玻利維亞、智利的小資探索祕境之旅

2016-05-04

Mobile01 原文

這趟南美旅行以秘魯為主軸，串起納斯卡線、庫斯科、印加聖谷與馬丘比丘。原文是早期發表在 Mobile01 的長篇遊記，這裡整理成作品集版本，保留旅行脈絡、拍攝亮點與精選照片。

  • 三國路線：秘魯、玻利維亞、智利
  • 環狀移動約 4000 km
  • 主題：印加文明、安地斯高原、古城與遺跡

從短時間規劃到南美長線旅行

行前準備時間很短，機票、簽證、轉機與內陸移動都需要快速決定。南美距離台灣很遠，飛行與轉機時間長，真正考驗的是路線取捨：在有限天數裡保留最值得拍攝與體驗的地點。

納斯卡、帕拉卡斯與沙漠綠洲

納斯卡線是旅程一開始的重要目標，搭小飛機觀景需要體力與抗暈準備。沿線也串起木乃伊、鳥島與 Huacachina 沙漠綠洲，形成秘魯海岸線與沙漠地貌的開場。

庫斯科與印加文明

庫斯科曾是印加帝國首都，也是前往馬丘比丘的核心城市。高海拔、強烈日照、西班牙殖民建築與印加石牆交錯，讓城市本身就是一段歷史剖面。

印加聖谷

印加聖谷包含 Chinchero、Maras Mines、Moray 與 Ollantaytambo 等地點。這段旅程從人文、農業實驗場、鹽田到古城要塞，展示印加文明如何在安地斯環境中生活與發展。

馬丘比丘與拍攝攻略

馬丘比丘是整趟旅程最重要的視覺高潮。清晨第一班車、雲霧、遊客流量與午後光線都會影響拍攝。作品集版保留這些經驗，方便之後整理成更完整的攝影筆記。
